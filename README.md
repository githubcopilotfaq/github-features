## Github Copilot 代码补全

| 功能名称 | 描述 | 支持的IDE |
|---------|------|------|
|[代码补全](https://docs.github.com/zh/copilot/how-tos/get-code-suggestions/get-ide-code-suggestions)|IDE中代码自动补全|VSCode,Jetbrains,Visual Studio,Eclipes,XCode|
|[下一行建议](https://docs.github.com/zh/copilot/how-tos/get-code-suggestions/get-ide-code-suggestions#navigating-and-accepting-next-edit-suggestions-1)|自动根据上下文提供下一行需要修改的代码建议|VSCode,Jetbrains,Visual Studio,Eclipes,XCode|

## Github Copilot Chat 功能

| 功能名称 | 描述 | 支持的IDE |
|---------|------|------|
|[自定义指令文件](https://github.com/orgs/githubcopilotfaq/discussions/23)|自定义指令文件，预先配置企业代码规范风格和项目背景上下文，确保Copilot生成的代码符合企业需求|VSCode,Jetbrains,Visual Studio,Eclipes,XCode|
|[计划模式](https://github.com/orgs/githubcopilotfaq/discussions/46)|计划模式帮助Copilot 在生成代码前进行规划，提升代码质量和一致性|VSCode,Jetbrains,Visual Studio,Eclipes,XCode|
|[自定义Agent模式](https://github.com/jacwu/github-materials/blob/main/github-copilot-lab/lab/cn/03.CustomAgent.md)|自定义Agent，用于创建企业特定功能的Agent|VSCode,Jetbrains,Visual Studio,Eclipes,XCode|
|[自定义聊天模式](https://github.com/orgs/githubcopilotfaq/discussions/18)|功能类似自定义Agent，此功能将逐渐被自定义Agent取代|VSCode,Jetbrains,Visual Studio,Eclipes,XCode|
|[使用MCP](https://github.com/orgs/githubcopilotfaq/discussions/34)|已PlayWright为例，介绍如何在Github Copilot 中使用MCP Server|VSCode,Jetbrains,Visual Studio,Eclipes,XCode|
|[SpecKit规范驱动开发](https://github.com/orgs/githubcopilotfaq/discussions/43)|使用Spec-kit实现规范驱动开发SDD|VSCode,Jetbrains,Visual Studio,Eclipes,XCode|
|[Copilot Cli](https://github.com/jacwu/github-materials/blob/main/github-copilot-lab/lab/cn/09.CopilotCLI.md)|Copilot 命令行功能|Windows,MacOS,Linux|
｜[代理会话](https://github.com/orgs/githubcopilotfaq/discussions/47)｜代理会话功能，支持在VSCode 中对多个Agent 会话进行管理｜VSCode｜
